## Repo structure

```
presentations/
├── README.md
├── index.html                          ← landing page listing all decks
└── <deck-folder>/
    ├── deck.html                       ← entry point (full navigable deck)
    ├── slide-01-*.html ... slide-NN-*.html
    ├── favicon.svg
    └── assets/                         ← photos, images
```

Each deck is **self-contained** in its own folder. New deck = new folder + add a row to the table above.

## How a deck works

- `deck.html` is the navigable entry — it loads each slide as an iframe and adds keyboard + edge-arrow navigation.
- Each `slide-NN-*.html` is also a **standalone** file you can open directly.
- Pure HTML/CSS/JS — no build step, no dependencies beyond Google Fonts.
- Works offline once cached. Works in any modern browser.

## Keyboard shortcuts (in `deck.html`)

| Key | Action |
|-----|--------|
| `→` `Space` `PageDown` | Next slide |
| `←` `PageUp` | Previous slide |
| `1`–`9` | Jump to slide |
| `F` | Toggle fullscreen |
| `H` | Hide / show navigation |
| `?` | Help overlay |
| `Esc` | Exit fullscreen / close help |
